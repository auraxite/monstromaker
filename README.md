# Монстродел

Карточная игра для 2–4 игроков: собери самого страшного монстра!

## Запуск

```bash
pip install .
monstromaker
```

Английский интерфейс:

```bash
monstromaker --lang en_US
```

Передать имена игроков сразу:

```bash
monstromaker --players Аня Боря Вася
```

Запустить из скрипта (командный файл):

```bash
monstromaker --players Аня Боря --script commands.txt
# или через перенаправление stdin:
monstromaker --players Аня Боря < commands.txt
```

Воспроизводимая раздача карт (фиксированное зерно):

```bash
monstromaker --seed 0 --players Аня Боря
```

## Правила (кратко)

На своём ходу:
1. Возьми одну карту (автоматически).
2. Сыграй одну карту — или нажми **[п]** для паса.

**Облик [ОБ]** — добавь часть тела монстру (★ навсегда).  
**Действие [ДЕ]** — разовый эффект.  
**Угроза [УГ]** — атака на противников.

Кто первым наберёт **15★** — победил. Если колода кончилась — у кого больше ★.

## Команды в игре

| Ввод | Действие |
|------|----------|
| `1`–`6` | Сыграть карту с таким номером |
| `п` или `p` | Пас (пропустить ход без карты) |
| `?` | Описание карты (программа спросит номер) |
| `д` | Подтвердить выбор |
| `н` или `n` / `0` | Отменить / вернуться назад |
| `Enter` | Передать ход / продолжить |
| `q` | Выйти из игры |
| `Ctrl-D` | Завершить программу |

## Формат командного файла (--script)

Файл `commands.txt` — один ввод на строку (пустая строка = Enter):

```
1

п


2
```

Пример — два игрока по 3 хода каждый (воспроизводимо):

```bash
monstromaker --seed 0 --players Аня Боря --script /tmp/demo.txt
```

Содержимое `/tmp/demo.txt` (пустая строка = Enter):

```
              ← Enter «начать»
п             ← Аня пасует (ход 1)
              ← Enter «передать ход»
п             ← Боря пасует (ход 2)
              ← Enter «передать ход»
5             ← Аня играет карту 5 (ход 3)
              ← Enter «передать ход»
5             ← Боря играет карту 5 (ход 4)
              ← Enter «передать ход»
п             ← Аня пасует (ход 5)
              ← Enter «передать ход»
п             ← Боря пасует (ход 6)
              ← Enter «передать ход»
q             ← выход
```

## Требования

- Python 3.10+
- `cowsay` (для финального отображения монстра)
- Linux / macOS
