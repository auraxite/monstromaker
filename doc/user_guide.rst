Руководство пользователя
========================

Правила игры
------------

Монстродел - карточная игра для 2–4 игроков (от 6 лет, 20–30 минут).

**Цель:** набрать больше всех звёздочек ★ у своего монстра.

**Начало:** каждый игрок получает 4 карты. Порядок хода перемешивается случайно.

**Ход:**

1. Взять одну карту с верха колоды (автоматически).
2. Сыграть одну карту из руки **или** нажать **п** - пас (нельзя пасовать два хода подряд).

Типы карт:

- **[ОБ] Облик** - часть монстра, остаётся перед игроком и даёт ★.
- **[ДЕ] Действие** - разовый эффект, после игры карта уходит в сброс.

**Конец игры:** колода закончилась или кто-то набрал **15★**.
Победитель - игрок с наибольшей страшностью.

Управление
----------

========  ==========================================
Клавиша   Действие
========  ==========================================
1–N       Сыграть карту с таким номером из руки
п         Пас (пропустить ход без игры карты)
?         Показать описание карты
========  ==========================================

После хода передайте клавиатуру следующему игроку.

Уровни страшности
-----------------

==========  =============
Звёздочки   Прозвище
==========  =============
0–3 ★       Пугало
4–6 ★       Страшилка
7–10 ★      Кошмарик
11+ ★       Ужастик
==========  =============

Карты (32)
----------

ОБЛИК (18 карт)
~~~~~~~~~~~~~~~

- **Крылья** ×2 - +2★
- **Рога** ×2 - +3★
- **Хвост** ×2 - +1★
- **Ядовитые когти** ×2 - +2★
- **Бронечешуя** ×2 - +2★, щит от Пекла
- **Три глаза** ×2 - +2★
- **Щупальца** ×2 - +3★
- **Клыки** ×2 - +2★
- **Жало** ×2 - +1★

ДЕЙСТВИЕ (14 карт)
~~~~~~~~~~~~~~~~~~

- **Лазер** ×2 - цель теряет 1 облик
- **Вор** ×2 - укради 1 облик у любого игрока
- **Камень усиления** ×2 - следующий облик +2★ дополнительно
- **Сделка** ×2 - возьми 2 карты, сбрось 1 из руки
- **Рёв ужаса** ×1 - все другие сбрасывают 1 карту из руки
- **Шпион** ×1 - тайно посмотри руку любого игрока
- **Жулик** ×1 - возьми любую карту из сброса
- **Пекло** ×2 - без Бронечешуи остальные теряют 1 облик
- **Экзамен по ОКам** ×1 - все сбрасывают 1 карту из руки
