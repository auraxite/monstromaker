Об игре
-------

**Монстродел** - карточная игра для 2–4 игроков, в которой каждый строит
своего монстра, добавляя ему части тела (крылья, клыки, щупальца и т.д.).
Побеждает тот, чей монстр окажется самым страшным!

Быстрый старт
-------------

.. code-block:: bash

   pip install monstromaker
   monstromaker

Запуск на английском языке:

.. code-block:: bash

   monstromaker --lang en_US
